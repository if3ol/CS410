# CS410 Academic Paper Recommender System

## Files
- **arxiv_utils.py** - Fetch papers from arXiv
- **recommender.py** - Unified recommender (TF-IDF + SciBERT + Hybrid)
- **cli.py** - Interactive command-line interface
- **example.py** - Programmatic usage example

## Quick Start
### Option 1: Interactive CLI
```bash
python cli.py
```
Features:
- Choose recommender (TF-IDF, SciBERT, or Hybrid)
- Search with natural language queries
- Give feedback (like/dislike papers)
- View your profile (liked/disliked papers)
- Reset profile
- Fetches 1000 papers automatically
### Option 2: Example Usage
```bash
python exmaple.py
```
